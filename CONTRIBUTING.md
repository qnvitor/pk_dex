# Guia de Contribuição

Obrigado por considerar contribuir com o projeto Pokédex com IA!

## Como Contribuir

### Reportando Problemas

Ao reportar um problema, por favor inclua:
- Versão do Python
- Sistema operacional
- Passos para reproduzir o problema
- Mensagens de erro completas (se houver)
- Screenshots (se aplicável)

### Sugerindo Melhorias

Sugestões são bem-vindas! Por favor:
- Descreva a melhoria proposta
- Explique por que seria útil
- Se possível, forneça exemplos de como funcionaria

### Enviando Código

1. **Fork o repositório**
2. **Crie uma branch** para sua feature (`git checkout -b feature/MinhaFeature`)
3. **Faça commit** das mudanças (`git commit -m 'Adiciona MinhaFeature'`)
4. **Push para a branch** (`git push origin feature/MinhaFeature`)
5. **Abra um Pull Request**

### Padrões de Código

- Use Python 3.13+ quando possível
- Siga o PEP 8 para estilo de código
- Adicione comentários explicativos quando necessário
- Mantenha funções pequenas e focadas
- Adicione tratamento de erros apropriado

### Estrutura do Projeto

- `pages/`: Páginas do Streamlit (sistema de páginas múltiplas)
- `src/`: Código fonte principal
- `scripts/`: Scripts utilitários
- `app/components/`: Componentes reutilizáveis do Streamlit

### Testes

Antes de enviar:
- Teste suas mudanças localmente
- Verifique se não quebrou funcionalidades existentes
- Teste em diferentes navegadores (se aplicável)

### Documentação

- Atualize o README.md se necessário
- Documente novas funcionalidades
- Mantenha comentários no código atualizados

## Perguntas?

Abra uma issue para discussão antes de fazer grandes mudanças.
